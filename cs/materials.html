Computer Science - Materials

For the best experience, the following materials will be nessary:

  • A laptop running Windows or OSX with a keyboard and the software below installed. Linux, Chromebooks, and Tablet Computers are highly discouraged.
  • A mouse, if you pefer it over a trackpad.

In addition to the above, the free software below should be downloaded onto your computer prior to a course:

  • A modern browser that supports HTML5, such as the latest versions of Google Chrome(recommended), Firefox, Safari, Opera, or Microsoft Edge.
  • The atom text editor, or any other text editor that supports syntax highlighting
  • The github desktop application.
